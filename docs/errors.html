Aurora REST API - Error Codes

400 Errors

Bad Request

Code Error Cause Fix
40001 Validation Error You have sent data that did not meet the validation requirements for a POST/PUT/PATCH request. Conform to the required data format and required fields.

401 Errors

Unauthorized

Code Error Cause Fix
40101 Unauthorized Request Unidentified branch request. Make sure that the branch_id you're sending in the html header is correct.
40102 Unauthorized Viewing Request Unauthorized view request of a resource. Make sure user have proper authorization for requested action.
40103 Unauthorized Modification Request Unauthorized modification request of a resource. Make sure user have proper authorization for requested action.

403 Errors

Forbidden

Code Error Cause Fix
40301 Blah Blah Blah

404 Errors

Not Found

Code Error Cause Fix
40401 No Resource Found There are no resource to return based on your URI request. Make sure you are using correct URI and valid parameters.

406 Errors

Not Accepted

Code Error Cause Fix
40601 Request Not Acceptable The request accept header does not allow application/json format. Add application/json in the request accept header.
40602 Request Origin Unknown The request header does not include a Branch. Add a valid Branch in the request header.

500 Errors

Internal Server Error

Code Error Cause Fix
50001 Database Connection Error
  • Improper connection settings
  • MySQL service not available
  • Database driver problem
 Make sure connection settings, MySQL service and database driver are properly configured
50002 Invalid SQL Query A data sent to the server has caused an unexpected error in an SQL Query. Contact API support and provide the error details.
50003 Internal Code Error An invalid instruction code is being run. Contact API support and provide the error details.
